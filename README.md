## Aplicativo de Teste

O aplicativo utilizado neste projeto não está incluído no repositório devido ao limite de tamanho do GitHub.

Baixe manualmente a versão mais recente do aplicativo de demonstração do WebDriverIO:

**Link:** https://github.com/webdriverio/native-demo-app/releases

Após o download, salve o arquivo conforme o sistema desejado:

- Para Android: `apps/native-demo-app.apk`
- Para iOS: `apps/native-demo-app.app`

Certifique-se de que o diretório `apps/` exista na raiz do projeto antes de executar os testes.
